# Lista de Exercícios I: Terminal e Git

> O objetivo deste exercício é conseguir consolidar e aprofundar os conceitos de utilização do
terminal e de cada um dos seus diferentes comandos.

### Exercício Prático
<p>1. Crie uma pasta chamada “arquivos”.</p>
<p>mkdir arquivos</p>

<p>2. Entre dentro desse diretório.</p>
<p>cd arquivos</p>

<p>3. Crie um arquivo chamado “teste.txt” e outro “hello.txt”.</p>
<p>touch teste.txt</p>
<p>touch hello.txt</p>

<p>4. Utilize o comando que usamos para ver o que temos dentro da pasta.</p>
<p>ls</p>

<p>5. Renomeie “hello.txt” para “teste2.txt”.</p>
<p>mv hello.txt teste2.txt</p>

<p>6. Renomeie “teste.txt” para “teste1.txt”.</p>
<p>mv teste.txt teste1.txt</p>

<p>7. Use o comando novamente para ver os arquivos que temos dentro da pasta, para ver se as alterações foram feitas.</p>
<p>ls</p>

<p>8. Exclua o arquivo “teste2.txt”.</p>
<p>rm teste2.txt</p>

<p>9. Use o comando para subir um diretório acima da pasta “arquivos” onde estamos trabalhando.</p>
<p>cd ..</p>

<p>10. Limpe a tela do terminal.</p>
<p>clear</p>
